
# Lvalue
  - 수정할수 있는 객체
  - 메모리를 실제로 할당
  - 호출할 수 있는 이름이 있음

# Rvalue
  - 수정할 수 없는 값
  - 메모리주소를 가지지않음
  - 일시적인 값


# Program Naming Convention
  - Camel Case : 첫글자는 소문자로 하되, 그뒤로 따라오는 단어들의 첫글자는 대문자 기입
    - exampleValue
  - Pascal Case : 모든 단어들의 첫글자를 대문자로 기입
    - ExampleValue
  - Kebab Case : 모든 단어를 -로 연결
    - example-value
  - Snake Case : 모든 단어를 _ 로 연결
    - exampe_value
    - EXAMPLE_VALUE



# 이동생성자 (Move Constructor)
